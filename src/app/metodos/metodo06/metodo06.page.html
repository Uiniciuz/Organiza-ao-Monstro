Autointerrogação

Essa técnica é indicada para estudantes do nível médio ou superior, pois como esses estudantes possuem uma grande bagagem esse método ajuda a gravar melhor o conteúdo. Há, e esse método é mais indicado para provas, conceitos, matérias teóricas já que é baseado na repetição de conceitos.

O método consiste em fazer perguntas para se mesmo enquanto está estudando, essas perguntas devem ser bem elaboradas, além de serem perguntadas e respondidas em voz alta, para melhor absorção do conteúdo.

Se preferir, pode-se repetir as perguntas e as repostas até elas estarem bem fixadas. Uma boa dica para elaborar as perguntas é imaginar contextos que possam cair em concursos e provas semestrai.

Um exemplo desse modo é quando uma pessoa ler sobre um tipo de vanguarda literária e, depois, perguntar para se mesmo quais são os principais autores da época e os títulos renomados, tentando se lembrar das obras que podem servir de exemplo.
