Resumos

Esse método é recomendado para pessoas que não gostão de utilizar mapas mentais por não saber desenhar ou outros motivos, mas gosta de aprender estruturando o conteúdo.

Nele você reúne os pontos principais de cada assunto, o que lhe permite lembrar de conteúdos extensos de forma rápida. Os resumos, devem ser sucintos, ou seja, não precisa escrever muito. Assim é bom estudar o conteúdo para ter um domínio antes de começa-lo. Lembre-se grifar nomes ou palavras chaves sempre é importante.

Algumas dicas:

  • Leia e releia o texto
  • Busque conceito, indicações e palavras-chaves
  • Escreva o texto com as suas próprias palavras
  • Organize as ideias principais
  • Deixe espaços para preencher conteúdos
  • Use post-it para observações
